How to Use

1. Installation & Setup

Install from Chrome Web Store. Pins the icon.
Then, name your "Browser Angel" in the setup page.

2. Daily Usage

Click the icon to see your tab count. Use 'Copy' to save for later, or wait for the 18:00 reminder.

3. Advanced

Set up Google Spreadsheet integration.
