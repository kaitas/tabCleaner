# 🧹 Tab Cleanup - The Browser Angel

**"あなたのブラウザに住む天使が、デジタルの平穏を取り戻します。"**

Tab Cleanup は、単なるタブ整理ツールではありません。これは、マッシブな分散システムによるゲーミフィケーションであり、デジタルライフを整えるための実験的な試みです。

## ✨ コンセプト

### 👼 The Browser Angel
この拡張機能には、実態のない「エージェント（天使）」が宿っています。
あなたがどれだけのタブを開き、どんな情報を摂取し、そしていつ休息を取るべきか... 天使は密かに記録し、優しくアドバイスします（v0.4〜）。

- **Scam Free / Spyware Free**:
  これはスパイウェアではありません。ユーザーの許可なく個人情報を外部サーバーへ送信することは**断じてありません**。
  天使はあなたのローカル環境（と、あなたが許可した安全なクラウドストレージ）にのみ存在します。

### 🎮 Distributed Gamification (分散ゲーミフィケーション)
私たちは、世界中のユーザーがどれだけタブを「断捨離」したかを、匿名化された統計として可視化する壮大なゲームを計画しています。
毎日のストリーク（継続記録）や、特定の条件達成によるバッジ獲得など、整理整頓を楽しむための仕掛けを用意します。

### 💸 Zero Operation Cost (運用コストゼロ)
このプロジェクトは、「作者が死んでも動き続ける」ことを目指して設計されています。
- **Serverless**: 重厚なバックエンドサーバーは持ちません。Cloudflare Workers と GitHub Pages を活用。
- **No Maintenance**: 可能な限り自動化され、永続的に無料または超低コストで運用可能なアーキテクチャを採用します。

## 🚀 機能 (Current & Roadmap)

### 基本機能 (v0.1 - v0.3)
- **📋 タブ一覧コピー**: 現在開いているタブをテキスト形式でクリップボードにコピー
- **📝 ブログ用出力**: Markdown形式でドメイン別に整理した一覧を生成
- **🚪 全閉じ＆記録**: 全タブを記録してから閉じる（Google Spreadsheet連携対応）
- **⏰ タイマー通知**: 18:00のお片付けリマインダー

### Next Phase (v0.4+)
- **🗣 Angels Name**: 天使に名前をつけてください。それがあなたのパートナーになります。
- **📊 Global Stats**: 匿名化されたデータにより、世界中の「断捨離総量」を可視化します。
- **🍅 Pomodoro**: 集中と休息のリズムを作るポモドーロタイマー。

## 🚀 インストール

### 開発版（ローカル）

1. このリポジトリをクローン
```bash
git clone https://github.com/kaitas/tab-cleanup-extension.git
cd tab-cleanup-extension
```

2. Chromeで `chrome://extensions` を開く

3. 右上の「デベロッパーモード」をON

4. 「パッケージ化されていない拡張機能を読み込む」をクリック

5. クローンしたフォルダを選択

## ⚙️ 設定

拡張機能のアイコン → 「設定」または右クリック → 「オプション」から設定画面を開けます。

### タイマー設定
- **警告時刻**: デフォルト 18:00
- **強制終了時刻**: デフォルト 21:00

### Google Spreadsheet連携（オプション）
1. Google Spreadsheetを新規作成
2. Apps Script（GAS）をデプロイ（`gas/Code.gs`参照）
3. 設定画面でSpreadsheetのURLを入力

## 🎯 使い方

### 日常の流れ

1. **日中**: 普通にブラウジング
2. **18:00（警告時刻）**: 通知が届く → 不要なタブを閉じる
3. **21:00（終了時刻）**: 自動で全タブを記録して閉じる
4. **翌日**: 履歴から昨日のタブを確認可能

### 手動操作

- **今すぐコピーしたい**: ポップアップ → 「タブ一覧をコピー」
- **ブログ記事にしたい**: ポップアップ → 「ブログ用に出力」
- **今すぐ閉じたい**: ポップアップ → 「全て閉じて記録」
